# Ping Pong scoreboard

## How to run 🚗

### NPM

1. clone the repository
2. ```bash
   cd repository_path
   ```
3. ```bash
   npm install
   ```
4. ```bash
   npm run build
   ```
5. ```bash
   npm start
   ```
6. Go to ➡️ [localhost:3000](http://localhost:3000)
